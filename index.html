Web制作 & 動画編集を学習中の初心者エンジニア

自己紹介

S.Nと申します。

現在、Web制作と動画編集の分野に興味を持ち、日々学習を進めている初学者です。

最近は、PHPとMySQLを用いて家計簿アプリを開発いたしました。
エラーや仕様の理解に苦戦しながらも、調べて試行錯誤を繰り返すことで、ひとつのアプリとして形にすることができました。
自分の手で「動くものを作れた」という経験は、大きな達成感と自信につながっています。

また、動画編集に関しては前職での業務経験を活かし、構成から字幕付けまでをすべて自身で行ったオリジナル作品を制作しました。
表現力や伝える力の向上を意識しながら、視聴者に楽しんでいただける内容づくりを心がけています。

まだ未熟な点も多いですが、「やってみたい」を形にできる力を身につけられるよう、今後も継続して学習を重ねてまいります。

制作物①：家計簿アプリ


この家計簿アプリは、PHPによるサーバー処理とMySQLによるデータ管理を活用し、支出の登録・一覧表示・合計計算・削除といった基本的な収支管理機能を実装したシンプルなWebアプリケーションです。

画面構成にはHTMLとCSSを使用し、XAMPPを用いたローカル開発環境で制作しました。
本アプリは、PHP・MySQL・HTML・CSSを使った初めての開発作品であり、学習初期の段階で取り組んだプロジェクトです。

仕様の理解やエラーへの対応には苦労しましたが、実際に手を動かして試行錯誤を重ねる中で、Webアプリケーションの基本的な仕組みを実践的に学ぶことができました。今後は、CSVファイルの取り込み機能やデザインの改善、さらに利便性を高める新機能の追加にも取り組み、より使いやすいアプリを目指していきたいと考えています。

使用技術一覧
  • PHP：データ処理や画面切り替え
  • MySQL：支出データの保存（DBとの連携）
  • HTML / CSS：画面構造の作成／見た目の調整
  • XAMPP：ローカル開発環境
  • Visual Studio Code：コードエディタ
  • Git / GitHub：バージョン管理・コードの公開

制作物②：YouTube用動画


動画編集スキルの習得を目的として、YouTube向けに複数の動画を制作いたしました。

編集には主に無料ツール（Microsoft Clipchamp）を使用し、カット編集・字幕の挿入・BGMの選定・テンポの調整など、視聴者にとって見やすく伝わりやすい構成を意識しています。

内容としては、キャンプ動画やゲーム動画、ショート動画を中心に制作しており、サムネイル作成なども含めて一人で完結させています。
現在も試行錯誤を重ねながら、より魅力的な映像表現を目指して取り組んでいます。
